# F-ing mosml

This repository contains an implementation of F-ing modules, written in Moscow ML.

## See also

- [elpinal/modules](https://github.com/elpinal/modules): F-ing modules in Haskell.
- [elpinal/modules-rs](https://github.com/elpinal/modules-rs): F-ing modules in Rust.

F-ing modules.
